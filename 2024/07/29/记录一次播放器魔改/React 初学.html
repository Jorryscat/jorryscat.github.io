React 初学

  1. 什么是 React

  2. react 是一个库 ，官方定义: The library for web and native user interfaces ( 用于构建用户界面的库，由Facebook在2013年开源 )

  3. 结合生态库构成框架 （ react， react-router，rudux，axios，babel，webpack）

  4. React的特点

  5. ==声明式编程== ：在声明式编程中，关注的是“做什么”，而不是“如何去做”。描述的是想要的结果，而框架或库负责实现细节。（ 以vue框架实现以及原生js为例）

// 声明式
<template>
  <div>
    <p>{{ message }}</p>
    <button @click="changeMessage">Change Message</button>
  </div>
</template>
<script>
export default {
  data() {
    return {
      message: 'Hello, Vue!'
    };
  },
  methods: {
    changeMessage() {
      this.message = 'Hello, World!';
    }
  }
};
</script>

//命令式: 在命令式编程中，您明确指示计算机如何执行操作，包括多步操作和详细的控制流程。您通常需要手动管理状态和用户界面。
例如，手动操作 DOM 来更新内容：

<script>
const button = document.querySelector('button');
const paragraph = document.querySelector('p');

let message = 'Hello, Vue!';
paragraph.textContent = message;

button.addEventListener('click', function() {
  message = 'Hello, World!';
  paragraph.textContent = message; // 手动更新 DOM
});
</script>
